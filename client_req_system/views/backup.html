Восстановление из бэкапа

⚠️
Внимание! Критическое действие
Восстановление из бэкапа перезапишет текущую базу данных.
Все текущие данные будут заменены данными из бэкапа.
Это действие необратимо!

После восстановления:

  • Сервер необходимо перезапустить
  • Все текущие сессии пользователей будут сброшены
  • Новые данные, созданные после бэкапа, будут утеряны
